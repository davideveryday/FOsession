# FOsession

Focus On Session is a simple and lightweight session stopwatch for VSCode.
Track your coding sessions directly from the status bar.

## Features
- Start session
- Pause/resume
- Reset timer
- Status bar integration

![FEATURES](/images/preview.gif)

## Commands

- `FOsession: Start Session`
- `FOsession: Pause Session`
- `FOsession: Resume Session`
- `FOsession: Reset Session`
- `FOsession: Stop Session`
